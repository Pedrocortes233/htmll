Ls tres pilares de la Web

Tim Berners-Lee es considerado el padre de la Web porque desarrolló los tres elementos básicos opara el funcionamiento de la Web:

HTML

HTTP

URL

HTML

HTML(HyperText Markup Language) es el lenguaje de marcado utilizado para crear y estructurar el contenido de las páginas web. Permite definir elementos como encabezados, párrafos, enlaces, imágenes y otros componentes multimedia.
